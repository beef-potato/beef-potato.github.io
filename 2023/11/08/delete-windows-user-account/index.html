delete windows user account
beef potato Lv3
  2023-11-08 15:16:44 2023-11-08 15:16   2023-11-08 15:26:41

室友要出自己的旧电脑，但是苦于不知道如何删除自己原来的 windows 账户， 在查阅一些网络资料后我帮他移除了账户。过程整理如下。

创建本地账户

https://support.microsoft.com/en-us/windows/create-a-local-user-or-administrator-account-in-windows-20de74e0-ac7f-3502-a866-32915af2a34d

根据步骤进行，其中需要同意windows的一个数据传输协议，跳过windows账户，指定名称，不需要密码即可创建 local account.

随后需要使用原来的管理员账户给这个local account 管理员权限来删除原来的管理员账户。

移除 user account

我是使用 windows 自带的 powershell 语句将其移除的。 参考这个网站.

首先以管理员身份打开powershell, 然后get-localuser 获取本地的所有用户

随后使用remove-localuser -name "xxx", 其中xxx为user的名称，需要注意部分user 是windows自带的user, enable状态是FALSE， 需要移除的account一般是TRUE。
